# MY VIM
my vim config

Requirements
--------------------------
  * Vim 7.3+

Installation (Mac OS X)
--------------------------

Clone repo:

    git clone git://github.com/zhenkunou/myvim $HOME/myvim

and run:

    cd ~/myvim

    ./setup.sh

--------------------------
