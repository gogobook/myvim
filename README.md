my vim config

Requirements
--------------------------
  * Vim 7.3+

Screenshots
--------------------------
![](https://raw.githubusercontent.com/zhenkunou/myvim/master/screenshots/myvim01.png)

Installation (Mac OS X)
--------------------------

Clone repo:

    git clone git://github.com/zhenkunou/myvim $HOME/myvim

and run:

    cd ~/myvim

    ./setup.sh
